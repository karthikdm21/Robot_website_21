# 🤖 Robot Shop

A modern and responsive e-commerce website for showcasing and selling robots designed for everyday life. The platform features a clean user interface where users can explore robots for household tasks, grocery shopping, security, cooking, and more.

## 🚀 Features

- Responsive design for desktop and mobile
- Modern and clean user interface
- Product showcase with robot categories
- Hero landing page
- Product cards with images
- Navigation bar and footer
- Smooth scrolling and interactive UI
- Easy to customize and extend

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- Tailwind CSS

## 🏃 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/robot-shop.git
```

Navigate to the project

```bash
cd robot-shop
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```
